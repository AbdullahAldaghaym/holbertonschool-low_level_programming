his project explores function pointers in C programming, demonstrating how to use pointers to functions for dynamic behavior and callbacks.

Learning Objectives
General
Understand what function pointers are and how to use them

Know what a function pointer exactly holds

Understand where a function pointer points to in the virtual memory

Master using function pointers for dynamic dispatch and callbacks

Project Structure
File	Description
0-print_name.c	Prints a name using a function pointer
1-array_iterator.c	Executes a function on each element of an array
2-int_index.c	Searches for an integer using a comparison function
3-calc.h	Header file for calculator program
3-op_functions.c	Arithmetic operation functions
3-get_op_func.c	Function selector for operations
3-main.c	Main program for calculator
100-main_opcodes.c	Prints its own main function's machine code
function_pointers.h	Header file containing all function prototypes
